Marching Cubes

Marching cubes is an algorithm which transforms a shape format into a polygon format. The animation below demonstrates the different stages of the marching cube animation in 2D.

2D Algorithm

  1. A Shape is created
  2. Overlay a grid on the shape
  3. Select the verticies inside the shape
  4. Generate mid-points based on vertex cases
  5. Draw lines between the generated points
